<strong>LamAPI - Label Matching API</strong> is a project that aims to provide an easy way to store and access <a href="https://www.wikidata.org/">Wikidata</a> dump.
Through an API specification, the tool allows Full-Text search over Wikidata and DBPedia and provides several services to retrieve data about entities. LamAPI can perform tasks like string classification, entity embedding representation.
